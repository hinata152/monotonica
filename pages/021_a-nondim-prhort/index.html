音速

文献 [1] のp. 297には a0=γRT0a_0 = \sqrt{\gamma R T_0}a0​=γRT0​​ という式が天下り的に出てきていて、これより前のページを見ても説明がされている様子はない（見落としてるだけかも）。

調べると別の文献 [2] が音速と流体の圧縮率の関係式を与えていた。

a2=(∂p∂ρ)s a^2 = (\frac{\partial p}{\partial \rho})_s a2=(∂ρ∂p​)s​

また理想気体の場合に成立する式

p=const⋅ργ p = const \cdot \rho^\gamma p=const⋅ργ

これより（等エントロピー流れを仮定の下で）見慣れた音速の式が得られる。

a2=∂p∂ρ=∂∂ρ(const⋅ργ)=const⋅γργ−1=γ(const⋅ργ)ρ=γpρ a^2 = \frac{\partial p}{\partial \rho} = \frac{\partial}{\partial \rho} (const \cdot \rho^\gamma) \\ = const \cdot \gamma \rho^{\gamma - 1} = \frac{\gamma (const \cdot \rho^{\gamma})}{\rho} \\ = \frac{\gamma p}{\rho} a2=∂ρ∂p​=∂ρ∂​(const⋅ργ)=const⋅γργ−1=ργ(const⋅ργ)​=ργp​

上式は理想気体の状態方程式 p=ρRTp = \rho R Tp=ρRT より以下のように書き換えることができる:

a2=γpρ=γRT a^2 = \frac{\gamma p}{\rho} = \gamma R T a2=ργp​=γRT

基準状態に対して以下の式が成立する。

a02=γp0ρ0=γRT0 a_0^2 = \frac{\gamma p_0}{\rho_0} = \gamma R T_0 a02​=ρ0​γp0​​=γRT0​

ここで VVV, TTT はいずれも無次元化された値である。 この式についても天下り的に出てきて詳細が書かれていないが、マッハ数の定義式より以下のように導出することができる。

無次元化

式 (5) を変形すると

p0=1γρ0a02 p_0 = \frac{1}{\gamma} \rho_0 a_0^2 p0​=γ1​ρ0​a02​

すると無次元化した基準圧力の値は1/γ1/\gamma1/γになる。

p0′=p0/ρ0a02=(1γρ0a02)/ρ0a02=1γ p_0' = p_0 / \rho_0 a_0^2 = (\frac{1}{\gamma} \rho_0 a_0^2) / \rho_0 a_0^2 = \frac{1}{\gamma} p0′​=p0​/ρ0​a02​=(γ1​ρ0​a02​)/ρ0​a02​=γ1​

(p′p0)=(ρ′ρ0)R(T′T0) (p' p_0) = (\rho' \rho_0) R (T' T_0) (p′p0​)=(ρ′ρ0​)R(T′T0​)

p′=ρ0RT0p0ρ′T′=ρ′T′∵p0=ρ0RT0 p' = \frac{\rho_0 R T_0}{p_0} \rho' T' = \rho' T' \qquad \because p_0 = \rho_0 R T_0 p′=p0​ρ0​RT0​​ρ′T′=ρ′T′∵p0​=ρ0​RT0​

したがって、基準圧力で無次元化した場合には圧力は密度と温度の無次元量の積で与えられる。 これはこれで便利。

状態方程式

理想気体の状態方程式

p=ρRT p = \rho R T p=ρRT

見慣れたこの式ではなく、以下のようないつもと異なる形で出てくることがある（1次元での表記）:

p=(γ−1)(e−12ρu2) p = (\gamma - 1)(e - \frac{1}{2} \rho u^2) p=(γ−1)(e−21​ρu2)

定圧比熱 cpc_pcp​ と定積比熱 cvc_vcv​ の間に成り立つ関係式

cp−cv=R c_p - c_v = R cp​−cv​=R γ≡cpcv \gamma \equiv \frac{c_p}{c_v} γ≡cv​cp​​

上2式より cv=1γ−1Rc_v = \frac{1}{\gamma - 1}Rcv​=γ−11​R を得る。 これを変形した R=(γ−1)cvR = (\gamma - 1) c_vR=(γ−1)cv​ を状態方程式に代入すると

p=ρRT=ρ(γ−1)cvT=ρ(γ−1)ϵ∵ϵ=cvT p = \rho R T = \rho (\gamma - 1) c_v T = \rho (\gamma - 1) \epsilon \\ \qquad \because \epsilon = c_v T p=ρRT=ρ(γ−1)cv​T=ρ(γ−1)ϵ∵ϵ=cv​T

ϵ\epsilonϵ は単位質量あたりの内部エネルギーであり、ρϵ\rho \epsilonρϵ で単位体積あたりの内部エネルギーになる。単位体積あたりの全エネルギーをeeeとすると、内部エネルギーとは全エネルギーから運動エネルギーを差し引いたものに等しい。すなわち:

p=(γ−1)ρϵ=(γ−1)(e−12ρu2) p = (\gamma - 1) \rho \epsilon = (\gamma - 1) (e - \frac{1}{2} \rho u^2) p=(γ−1)ρϵ=(γ−1)(e−21​ρu2)
